Det här gäller i korthet

Offroad Bumpis behandlar personuppgifter för att kunna hantera bokningar, svara på frågor, administrera betalningar och sköta uthyrningen på ett tryggt och tydligt sätt.

  • ✓Uppgifter används bara när det finns ett tydligt syfte.
  • ✓Känsliga uppgifter ska inte skickas in om de inte behövs.
  • ✓Personuppgifter sparas inte längre än nödvändigt.
  • ✓Personuppgifter säljs inte vidare.
Personuppgiftsansvarig

Offroad Bumpis drivs av Erica Nordlöf. Verksamheten ansvarar för hur personuppgifter hanteras.

Kontakt: kontakt@offroadbumpis.se

Vilka uppgifter som kan samlas in

När du bokar, kontaktar Offroad Bumpis eller skickar in ett formulär kan följande uppgifter behandlas:

  • ✓Namn
  • ✓E-postadress
  • ✓Telefonnummer om du lämnar det
  • ✓Ort, adress eller plats för leverans eller upphämtning
  • ✓Bokningsdatum, paket, tillval och meddelanden du själv skickar
  • ✓Faktura- eller företagsuppgifter om bokningen gäller företag, skola, förening eller fakturabokning
  • ✓Uppgifter du lämnar via mejl, bokningsverktyg eller kontaktformulär
Varför uppgifterna används

Personuppgifter används för att kunna:

  • ✓Ta emot och hantera bokningar
  • ✓Skicka bekräftelser, praktisk information och svar på frågor
  • ✓Hantera betalning, faktura och bokföringsunderlag
  • ✓Följa upp bokningar, ändringar, avbokningar eller eventuella skador
  • ✓Hantera kontaktförfrågningar
  • ✓Skydda verksamheten mot spam, missbruk och oseriösa förfrågningar
Rättslig grund

Personuppgifter behandlas främst för att kunna fullgöra en bokning eller ett avtal, till exempel när du bokar bumperballs, begär information eller behöver återkoppling kring en förfrågan.

Vissa uppgifter kan också behandlas för att uppfylla rättsliga skyldigheter, till exempel bokföring. Uppgifter kan även behandlas utifrån ett berättigat intresse, till exempel för att svara på kontaktförfrågningar och hålla ordning på verksamheten.

Formulär och e-post

Om du skickar in uppgifter via formulär på sidan eller via e-post används uppgifterna för att kunna svara dig, hantera din bokning, skicka information inför uthyrning eller följa upp praktiska frågor.

Skicka inte känsliga personuppgifter som inte behövs. Bumperballs behöver datum och plats, inte hela Netflix-dokumentären.

Bokning och betalning

Vid bokning kan personuppgifter behandlas för att reservera datum, hantera kontakt före och efter uthyrning, skicka dokumentation, bekräftelser och administrera betalning eller faktura.

Om bokningen sker via faktura används uppgifterna också för att skicka faktura och följa upp betalning före utlämning. Uppgifter kan även behöva sparas som del av bokföring, kundadministration och underlag kopplat till bokningen.

Dokumentation vid överlämning, återlämning och skada

Vid uthyrning kan uppgifter kopplade till bokningen användas för att dokumentera överlämning, återlämning och eventuella avvikelser.

Bilder eller video kan tas om det behövs för att dokumentera utrustningens skick, skada, saknade delar eller annan avvikelse. Detta används för administration, uppföljning och eventuell tvist.

Tredje parter och externa tjänster

Offroad Bumpis kan använda externa tjänster för till exempel bokning, formulärhantering, e-post, betalning, fakturering, webbpublicering och andra administrativa funktioner som behövs för att verksamheten ska fungera.

De tjänsterna behandlar i så fall uppgifter i den mån det behövs för att tjänsten ska fungera. Personuppgifter säljs inte vidare till andra.

Cookies och statistik

Webbplatsen kan använda tekniska lösningar som behövs för att sidan ska fungera. I vissa fall kan enkel statistik användas för att förstå hur sidan används och förbättra informationen.

Om externa tjänster används, till exempel bokningsverktyg eller formulärverktyg, kan de ha egna tekniska lösningar och egna villkor för hur deras tjänst fungerar.

Hur länge uppgifterna sparas

Uppgifter sparas inte längre än nödvändigt. Boknings- och fakturauppgifter kan behöva sparas så länge det krävs enligt lag, till exempel bokföringsregler.

Kontaktförfrågningar och formulär rensas när de inte längre behövs för kontakt, uppföljning eller administration.

Hur uppgifterna skyddas

Rimliga tekniska och administrativa åtgärder används för att skydda uppgifter mot obehörig åtkomst, förlust eller missbruk.

Endast uppgifter som behövs för verksamheten ska hanteras. Mindre data, mindre trassel. Det är bra governance även i barnkalasbranschen.

Dina rättigheter

Du har rätt att kontakta Offroad Bumpis om du vill:

  • ✓Få veta vilka personuppgifter som finns sparade om dig
  • ✓Begära rättelse om något är fel
  • ✓Begära att uppgifter tas bort när de inte längre behövs
  • ✓Invända mot viss behandling

Du kan också lämna klagomål till Integritetsskyddsmyndigheten om du tycker att personuppgifter hanteras fel.

Viktigt om känsliga uppgifter

Skicka inte känsliga personuppgifter via mejl eller formulär om det inte är absolut nödvändigt. Skriv hellre kort och praktiskt: datum, ort, paket, kontaktuppgifter och eventuell leveransinfo.

Frågor om integritet?

Kontakta Offroad Bumpis om du vill veta vilka uppgifter som finns sparade, begära rättelse eller ställa en fråga om personuppgifter.
