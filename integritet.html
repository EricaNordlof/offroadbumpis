Integritetspolicy

Här står hur Offroad Bumpis hanterar personuppgifter när du bokar, skickar mejl, använder formulär eller kontaktar mig på annat sätt.

  • ✓Uppgifter används för bokning, kontakt, betalning och administration.
  • ✓Personuppgifter säljs inte vidare.
  • ✓Du kan fråga vilka uppgifter som finns sparade.
  • ✓Du kan be om rättelse eller radering när det är möjligt.

Integritet i kort format

Här är det viktigaste först. Detaljerna ligger under varje rubrik, så sidan slipper bli en juridisk vägg.

Det här gäller i korthet

Offroad Bumpis behandlar personuppgifter för att kunna hantera bokningar, svara på frågor, ta betalt, skicka faktura och sköta uthyrningen på ett tryggt sätt.

  • ✓Uppgifter används bara när det finns ett tydligt syfte.
  • ✓Skicka inte känsliga uppgifter om de inte behövs.
  • ✓Personuppgifter sparas inte längre än nödvändigt.
  • ✓Personuppgifter säljs inte vidare.
Vem ansvarar för uppgifterna?

Offroad Bumpis drivs av Erica Nordlöf via Nordlöf Nordic. Offroad Bumpis ansvarar för hur personuppgifter hanteras.

Kontakt: kontakt@offroadbumpis.se

Vilka uppgifter kan samlas in?

När du bokar, skickar mejl eller fyller i formulär kan de här uppgifterna hanteras:

  • ✓Namn.
  • ✓E-postadress.
  • ✓Telefonnummer om du lämnar det.
  • ✓Ort, adress eller plats för leverans eller hämtning.
  • ✓Bokningsdatum, paket, tillval och meddelanden du själv skickar.
  • ✓Faktura- eller företagsuppgifter om du bokar via företag, skola eller förening.
  • ✓Uppgifter du lämnar via mejl, bokningsverktyg eller kontaktformulär.
Varför används uppgifterna?

Uppgifterna används för att kunna:

  • ✓Ta emot och hantera bokningar.
  • ✓Skicka bekräftelse och praktisk information.
  • ✓Svara på frågor.
  • ✓Hantera betalning, faktura och bokföring.
  • ✓Följa upp ändringar, avbokningar, skador eller saknade delar.
  • ✓Skydda verksamheten mot spam och oseriösa förfrågningar.
Rättslig grund

Uppgifter behandlas främst för att kunna genomföra en bokning eller svara på en fråga. Alltså: du kontaktar mig, jag behöver kunna svara och hantera det du frågar om.

Vissa uppgifter måste också sparas enligt lag, till exempel bokföringsunderlag. Ibland används uppgifter för ett berättigat intresse, till exempel för att hålla ordning på bokningar och kontakt.

Formulär och mejl

Om du skickar uppgifter via formulär eller mejl används de för att kunna svara, hantera bokningen eller skicka praktisk information.

Skicka inte känsliga personuppgifter om de inte behövs. Bumperballs behöver datum, plats och kontaktuppgifter. Inte hela livsarkivet.

Bokning och betalning

Vid bokning kan uppgifter användas för att reservera datum, kontakta dig, skicka bekräftelse, skicka dokument och hantera betalning.

Betalning kan ske med kort eller Klarna via betalningslänk, Swish, faktura eller bankgiro/banköverföring.

Vid faktura används uppgifterna för att kunna skapa och följa upp fakturan. Fakturauppgifter kan behöva sparas enligt bokföringsregler.

Dokumentation vid överlämning, retur och skada

Vid uthyrning kan uppgifter kopplade till bokningen användas för att dokumentera överlämning, återlämning och eventuella avvikelser.

Bilder eller video kan tas om det behövs för att visa utrustningens skick, skada, saknade delar eller annan avvikelse. Det används för administration, uppföljning och eventuell tvist.

Tredje parter och externa tjänster

Offroad Bumpis kan använda externa tjänster för bokning, formulär, mejl, betalning, faktura, webbpublicering och annan administration.

De tjänsterna kan behandla uppgifter i den mån det behövs för att tjänsten ska fungera. Personuppgifter säljs inte vidare.

Cookies och statistik

Webbplatsen kan använda tekniska lösningar som behövs för att sidan ska fungera. Enkel statistik kan också användas för att förstå hur sidan används och förbättra informationen.

Externa tjänster, till exempel bokningsverktyg eller formulärverktyg, kan ha egna tekniska lösningar och egna villkor.

Hur länge sparas uppgifterna?

Uppgifter sparas inte längre än nödvändigt. Boknings- och fakturauppgifter kan behöva sparas enligt lag, till exempel bokföringsregler.

Kontaktförfrågningar och formulär rensas när de inte längre behövs för kontakt, uppföljning eller administration.

Hur skyddas uppgifterna?

Rimliga tekniska och administrativa åtgärder används för att skydda uppgifter mot obehörig åtkomst, förlust eller missbruk.

Bara uppgifter som behövs ska hanteras. Mindre data, mindre trassel. En bra regel även när verksamheten handlar om barn som studsar runt i plastbollar.

Dina rättigheter

Du kan kontakta Offroad Bumpis om du vill:

  • ✓Veta vilka personuppgifter som finns sparade om dig.
  • ✓Rätta uppgifter om något är fel.
  • ✓Be att uppgifter tas bort när de inte längre behövs.
  • ✓Invända mot viss behandling.

Du kan också kontakta Integritetsskyddsmyndigheten om du tycker att personuppgifter hanteras fel.

Viktigt om känsliga uppgifter

Skicka inte känsliga personuppgifter via mejl eller formulär om det inte är absolut nödvändigt. Skriv hellre kort och praktiskt: datum, ort, paket, kontaktuppgifter och eventuell leveransinfo.

Ingen försäljning av uppgifter

Offroad Bumpis säljer inte personuppgifter vidare. Uppgifter används för att kunna driva uthyrningen, svara på frågor, hantera bokning, betalning, faktura och praktisk administration.

Det är inte mer dramatiskt än så. Bumperballs ska studsa. Kunddata ska inte studsa runt.

Frågor om integritet?

Kontakta Offroad Bumpis om du vill veta vilka uppgifter som finns sparade, be om rättelse eller fråga något om personuppgifter.
